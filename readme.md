## General info

This project is simple backend for portfolio site.

## Technologies

Project is created with following packages:

- axios
- bcryptjs
- cors
-  dotenv
-  express
-  express-jwt
-  jsonwebtoken
-  mongoose
-  multer
-  nodemailer

## Setup

To run this project, install it locally using npm:

```
$ npm install
$ npm start
```
